How to Use the Cloudflare Worker Script in Webflow

This guide explains how to integrate the Cloudflare Worker-hosted JavaScript into Webflow.

1. Add the Script to Your Webflow Project

Include the following script inside the <head> or before the closing </body> tag of your Webflow project:

<script src="https://yourdomain.com/script.js" defer></script>

2. How the Script Works

There are two ways to use this script in Webflow:

Using Webflow Tabs

If you are using Webflow Tabs, use the ctmk-tab-link attribute to define the target URL:

<a ctmk-tab-link="/about">Go to About</a>

When clicked, the script will navigate to the URL specified in the ctmk-tab-link attribute.

Using Regular Links or Div Links

If you are using normal links or div links instead of Webflow Tabs, you only need to use the ctmk-child and ctmk-menu attributes:

<div ctmk-menu>
    <a ctmk-child href="/services">Services</a>
    <a ctmk-child href="/contact">Contact</a>
</div>

These elements will work as regular navigation links.

3. Adding Active States

The script automatically adds an active class to elements inside a menu:

<div ctmk-menu>
    <a ctmk-child ctmk-tab-link="/services">Services</a>
    <a ctmk-child ctmk-tab-link="/contact">Contact</a>
</div>

If the current page matches the ctmk-tab-link value, it adds the ctmk-active class to highlight the active link.

4. Testing

Publish your Webflow project and click the links to test the functionality!
